# clair plugin (skeleton)

This is a skeleton scaffold for the clair plugin. The product vision is the single
source of truth in [docs/product.md](../docs/product.md): repo-level ambient awareness
via progressive disclosure — a clair discloses a live event the way a skill discloses
pre-defined context. No commands, hooks, or MCP server are wired up yet; the features
described in the vision are not implemented.
